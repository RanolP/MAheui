# MAheui
MAheui\[Modularized Aheui/마희\]는 모듈화된 아희에 대한 명세입니다.
마희 관련 프로그램·라이브러리는 다음 명세를 꼭 지켜야 합니다.

**주의**: 이 프로젝트는 심심해서 시작한거니, 흥미를 잃으면 금방 죽습니다.
